# Лабораторная работа №2: JIT-компиляция регулярных выражений

[Актуальная версия условия](https://sysprolang.b-andrew.ru/)

Run tests and benchmarks using benchmark harness (RegexApp):

```shell 
./gradlew run
```

Run only benchmarks using benchmark harness (RegexApp):

```shell
./gradlew runBenchmarks
```

Run only tests using JUnit (this is what GitHub pipelines trigger):

```shell
./gradlew test
```
